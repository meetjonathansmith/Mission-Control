# SC DORMANT PURGE - VA INSTRUCTIONS
**Date:** February 18, 2026  
**Execute:** ASAP

## Objective
Identify and move dormant agents to stop wasting send capacity on non-responders.

## Criteria (Non-Negotiable)
- **8+ text messages sent** in the last 90 days
- **Zero replies** (lifetime)
- Move to "Dormant" list
- Stop all future sends

## Step-by-Step Instructions for SmartContact

### Step 1: Create the Filter
1. Log into SmartContact
2. Go to **Contacts** → **Advanced Filter**
3. Set the following criteria:
   - **Messages Sent:** ≥ 8
   - **Date Range:** Last 90 days
   - **Replies Received:** = 0
   - **Status:** Active (not already dormant)

### Step 2: Export the List
1. Click **Export** (CSV format)
2. Save as: `SC-Dormant-Purge-Feb18-2026.csv`
3. Note the total count

### Step 3: Review Before Moving
**SANITY CHECK:**
- Open the CSV
- Spot-check 10-15 contacts
- Verify they actually have 10+ sends and zero replies
- If anything looks wrong, STOP and notify Jonathan

### Step 4: Move to Dormant List
1. Select all contacts from the filter
2. **Bulk Action** → **Move to List**
3. Select/Create list: **"Dormant - 8+ No Reply"**
4. Confirm the move

### Step 5: Stop All Campaigns
1. Go to **Campaigns** → **Active Campaigns**
2. For each campaign (MD, NOVA, Richmond, VA Beach):
   - Edit campaign settings
   - **Exclude List:** Add "Dormant - 8+ No Reply"
3. Save all campaigns
