## SECTION 1B: BOT INTEGRATION WORKFLOW

### The Problem Bot Solves

**Current bottleneck (Feb 2026):**
- LM spends 8-10 hours/day posting properties + qualifying
- Manual work = capacity ceiling at ~150-180 off-markets/month
- Jonathan steps in when LM gets backlogged (18+ off-markets/day for 3 days)

**What changes when bot goes live:**

### Workflow BEFORE Bot

1. **Agent responds** with off-market property
2. **VA logs** agent response in SC Campaign Log
3. **LM manually posts** property to GHL (5-10 min per property)
4. **LM manually qualifies** property (basic filters: price, condition, market)
5. **LM underwrites** qualified deals (MAO formula, comps, strategy)
6. **LM or Jonathan** communicates with agent for offer
7. **Jonathan closes** ratified deals

**Time per off-market:** 15-20 minutes (posting + qualifying + underwriting)  
**LM capacity:** 180 off-markets/month max (12-15/day sustainable)

---

### Workflow AFTER Bot

1. **Agent responds** with off-market property
2. **Bot automatically posts** property to GHL (30 seconds)
3. **Bot automatically qualifies** property using preset filters:
   - Price range check
   - Market/zip code validation
   - Property type filter
   - Distress signals present?
   - Basic math checks (estimated value, repair costs)
4. **Bot surfaces ONLY qualified deals** to LM (junk filtered out)
5. **LM underwrites** qualified deals (MAO formula, comps, strategy)
6. **LM or Jonathan** communicates with agent for offer
7. **Jonathan closes** ratified deals

**Time per off-market (qualified):** 5-10 minutes (underwriting + communication only)  
**LM capacity:** 300+ off-markets/month (bot handles 70-80% of grunt work)

---

### Bot Impact on Capacity

**Current (no bot):**
- 150-180 off-markets/month = LM capacity ceiling
- LM: 8-10 hours/day on posting + qualifying
- Jonathan: Steps in when backlogged

**With bot (Month 2 of 90-day plan):**
- 200+ off-markets/month = achievable
- LM: 4-5 hours/day on underwriting + agent communication
- Jonathan: Only touches high-signal closes (>$50K or complex zoning)

**Bot unlocks volume scale without adding headcount.**

### What To Do at Each Warning Level

**Tier 1 (Minor Fatigue):**
1. Refresh message variants (introduce 2-3 new templates)
2. Extend rest period by 2 weeks (from 8 to 10 weeks)
3. Test different send times (shift from morning to afternoon)
4. Reduce send rate by 25% (from 6/min to 4.5/min for MD)

**Tier 2 (Moderate Fatigue):**
1. Pause campaign for 4-6 weeks (mandatory rest)
2. Complete message overhaul (retire all old templates, write fresh angles)
3. Segment list: Remove agents with 20+ touches, move to dormant
4. Test small batch (500 agents) with new message before full relaunch
5. When relaunching: Slower cadence (2-touch-rest instead of 3-touch)

**Tier 3 (Severe Exhaustion):**
1. STOP all campaigns in that market immediately (12-week rest minimum)
2. Audit for spam flagging (see Section 6)
3. Re-scrape list (add fresh agents, remove bad data)
4. Overhaul entire strategy: New message angles, new sender names, new approach
5. Consider market replacement (if exhaustion can't be reversed, retire market)

---

### List Recovery Protocol

**If a market hits Tier 3 exhaustion:**

**Month 1-3: Full Rest**
- Zero contact (agents get a break)
- Refresh list: Add new agents from MLS updates, remove outdated contacts
- Message development: Write 5-7 completely new templates (different angles than anything sent before)

**Month 4: Small Batch Test**
- Test 1,000 agents with new message
- Monitor response rate, delivery rate, opt-out rate
- Goal: >2% response rate on test batch
- If successful: Proceed to Month 5
- If failed: Extend rest another 3 months

**Month 5: Controlled Relaunch**
- Launch to 25% of list (5,000-7,000 agents)
- 2-touch-rest cycle (initial + 1 follow-up only)
- Monitor closely for 4 weeks
- If response rate sustains >2%: Proceed to Month 6
- If drops below 2%: Market may be permanently exhausted

**Month 6: Full Relaunch**
- Scale to full list
- 3-touch-rest cycle resumes (if response rate healthy)
- Long-term: Maintain 10-12 week rest periods to prevent re-exhaustion

**Recovery success rate: ~60-70%** if rest period and message overhaul are executed properly.

**If recovery fails:** Market is exhausted. Retire list or reduce to 1 touch every 6 months (very low maintenance).

### Proactive Exhaustion Prevention

**Don't wait for Tier 3. Prevent it:**

1. **Mandatory rest periods** (built into campaign calendar, non-negotiable)
2. **Message rotation** (never send same message to same agent twice)
3. **Lifetime touch limits** (max 30 touches per agent over 3 years)
4. **Fresh list injection** (add 500-1,000 new agents quarterly)
5. **Segmentation** (warm agents separated from cold, treated differently)
6. **Seasonal adjustment** (lighter touch in slow seasons)
